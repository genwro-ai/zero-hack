Baselines and Data Generation

A practical starting plan for the Industrial AI track: generate valid symbolic process sequences, keep the first data pipeline simple, and compare a clear ladder of baselines before adding richer supervision.

Python 3.12 uv symbolic sequences baseline-first

Dataset Policy

1

Generate valid sequences

Use the provided generator directly. Start with 5k per family, then scale to 20k.

2

Validate and dedupe

Accept only sequences that pass the rule checker. Remove exact duplicates.

3

Split deterministically

Use 80 / 10 / 10 splits and make sure exact sequences do not cross split boundaries.

No coverage selector at the start. The existing generated data already shows good uniqueness, sane length ranges, and reasonable optional-branch coverage.
